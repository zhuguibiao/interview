flex-grow：

计算flex-grow大小，方法如下：

1、先计算剩余空间多少

2、看所有子元素的flex-grow总和为多少，然后分成多少等分。

3、算出结果为 = 设置的width + 当前元素flex-grow的值 * 每等分

带入下面demo及：

left的width = 300px + 1 * ((600-300-200) / (1 + 2)) = 333.33

right的width = 200px + 2 * ((600-300-200) / (1 + 2)) = 266.67

flex-shrink

计算flex-shrink的大小，方法如下：

1、先计算超出的空间值为多少

2、算出总权重为： 2 * 500+ 1 * 400 = 1400。

3、两个元素分别收缩：

300 * (2(flex-shrink) * 500(width) / 1400) = 214.28

300 * (1(flex-shrink) * 400(width) / 1400) = 85.72

4、三个元素的最终宽度分别为：

500 - 214.28 = 285.72

400 - 85.72 = 314.28
