# 操作规范

## 常用命令
```
git branch

git checkout -b test develop

git chekcout develop

git branch -d test

```

## 分支规范

- 主分支
  - 生产发布分支：production
  - 预发布发布分支：develop

- 临时性分支
  - 功能性：feature-*
  - bug：fixbug-*
